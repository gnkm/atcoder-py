## Usage

### Create virtual env

```
anyenv install pyenv
pyenv install 3.8.13
poetry env use $HOME/.anyenv/envs/pyenv/versions/3.8.13/bin/python
```

### Prepare

```
poetry install
poetry shell
```

### Execute

```
python xxx.py < input.txt
```

### Exit virtual env

```
exit
```
