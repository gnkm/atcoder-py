## Usage

### Create virtual env

```
anyenv install pyenv
pyenv install pypy3.7-7.3.4
poetry env use $HOME/.anyenv/envs/pyenv/versions/pypy3.7-7.3.4/bin/python
```

### Prepare

```
poetry shell
poetry install
```

### Execute

```
python xxx.py < input.txt
```

### Exit virtual env

```
exit
```
